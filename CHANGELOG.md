# CHANGELOG

## v0.1.0 - Unreleased

### 2025-04-12

- Added labels to posts & profiles

### 2026-04-08

- Forward/Back history navigation in the app rail/navigation, and thread drawer
- Added theme control (light/dark/system) with proper light theme support
- Help section with keyboard shortcuts and usage help

### 2026-04-07

- Image (single or gallery) & video player with blob downloading
- Group notifications by reason & post, with clickable links to subjects and posts
- Added dedicated post/thread view.

### 2026-04-06

- Post drafts in composer, with account scoped list, autosave and keyboard shortcuts.
- Add favicons to lexicon collections in the protocol explorer.

### 2026-04-05

- Make embedding model/semantic search opt-in with a dedicated preflight screen.

### 2026-04-04

- Add "Danger Zone" to clear cache & app data and reset the application
- Profile search with autocomplete/typeahead
- Hashtag and filterable post search for posts fetched through the BlueSky API

### 2026-03-31

- Multicolumn views with independent scrollable columns for feeds, search results, and explorer panels
  - DM/Messages column with privacy blur
- Profile view with multi-tab view for posts, followers, following, and activity. Includes a parallax
  header/cover image and dynamic sticky header.

### 2026-03-30

- Profile context/diagnostics using the [Constellation](https://constellation.microcosm.blue) API for
  backlinks
  - Blocking, Blocked by, Lists
- Dedicated Settings screen
  - Log viewer with level filtering
  - Theme and appearance settings
  - Exportable data

### 2026-03-29

- Polled notifications, with tabbed view for near real-time interaction updates.
- [pdsls](https://pds.ls) inspired AT Protocol explorer
- Local & network post search via vectors & embedding, and xrpc, respectively with tabbed panel for Saved & Liked posts

### 2026-03-28

- Multi-account support and OAuth-based, loopback authentication using [jacquard](https://tangled.org/nonbinary.computer/jacquard)
- Home screen/timeline view with saved feeds and customizable feed filters and preferences.
  - Feed management (reorder, pin/unpin) with full list of feeds.
